# lectureNotes

This repository contains collected notes organized by theme to make them easier to browse.

Top-level folders (themes)

- `programming/` — languages, frameworks, and fundamentals
- `devops-cloud/` — cloud, infra, monitoring, ops-tools
- `security/` — pentest, enterprise security, crypto
- `data-analytics/` — databases, data platform, ML
- `networking/` — networking notes
- `math-cs-fundamentals/` — math and CS foundations
- `architecture-design/` — system design and architecture
- `personal-languages-and-others/` — language study notes
- `other/` — miscellaneous notes

Each folder contains a `README.md` index. Browse the folders for detailed notes.
